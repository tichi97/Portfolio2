Tweet Analyzer

A web app that analyzes the sentiment of tweets by user or trend

This app allows users to either enter a twitter handle or a topic and returns a visual represntation of the sentiment of the tweets under that specific handle or topic. The tweets are categorized as either Positive, Negative or Neutral.

I did this by connecting with the Twitter API through the tweepy library. This allowed me to iterate through tweets and process the text in each tweet using the TextBlob to determine the tweet's sentiment.

Technologies used:
  • Python - Flask
  • Twitter API
  • HTML/CSS
  • Data visualization

Find My Tutor

A web app that connects tutors to students

This app allows tutors to create profiles describing the areas they tutor in so that interested students can connect with them.

When tutors create profiles, their information is stored in an SQLite database. Students can then visit the site (without the need to log in/ create an account) and browse through the available tutors. If they are interested in a tutor, they can click on the posting to find out more about the tutor and proceed to contact them through the available contact section.

Tutors must create an account in order to post their availability to tutor. They can also carry out the common user account features such as editing information, uploading a profile picture, editing/deleting tutoring posts that they have created, etc

Technologies used:
  • Python - Flask
  • HTML/CSS
  • MySQL
  • SMTP

Connect Four

A javascript implementation of the popular children's game Connect Four

This app allows two players to play connect four. There are two player colors: red and yellow. Each click on the screen results in a chip 'dropping' onto the board. Once there are four of the same color in a row in any direction, that color wins.

Technologies used:
  • HTML/CSS
  • Javascript
